Current Projects

I'm using my time off from teaching this summer to deep-dive into some of my programming interests. One project that's been successful already is my work on updating my reverb plugin — RS Algorithmic Verb, which you can download on GitHub. It's a collection of many different reverb algorithms, including modeling classic hardware, and some experiments with feedback delay networks.

I'm also finally learning Rust! I've wanted to for a while, and I'm finding it more approachable than I'd worried it might be. So far I'm working through The Rust Programming Language, as well as practicing some of my skills doing embedded programming on the BBC micro:bit microcontroller. I'm really enjoying the language, and I hope to get some audio plugins written in it.

One music project I'm currenly working on is a trio for oboe, percussion, and MIDI keyboard, tentatively titled Forget your name. I wrote about some of the sound design here and here, and while it has been going slowly, I've been happy with some of the sounds I've made.

I've also been able to do some work on my website. I added webmentions, added a reaction button to my blog posts (writeup forthcoming), and added dropdowns within the menu to make it easier to add more pages (such as this now page!).

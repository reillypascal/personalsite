Current Projects

I've been extra busy this spring semester at Geneseo—I taught a music business class for the first time, and that required a lot of lesson prep each week, since I needed to make all the lesson materials from scratch. As a result, I haven't been able to work on personal projects as much as I would like.

One music project I have been able to work on is a trio for oboe, percussion, and MIDI keyboard, tentatively titled Forget your name. I wrote about some of the sound design here and here, and while it has been going slowly, I've been happy with some of the sounds I've made.

I've also been able to do some work on my website. I added webmentions, added a reaction button to my blog posts (writeup forthcoming), and added dropdowns within the menu to make it easier to add more pages (such as this now page!).

Here are some projects I'm considering for the summer:

  • I want to add a "uses" page and a "notes" feed to this site. The "notes" page will be kind of like a Mastodon feed—short thoughts, webmention-based interactions, and other things I don't want to put in my blog.
  • I plan to resume making VST/AU plugins. I use C++/JUCE, and I would like to both update my reverb plugin and make some new ones.
  • For Christmas, my partner's parents gave me the book Designing Audio Objects for Max/MSP and Pd by Eric Lyon. It's in C, not C++, although the examples are straightforward enough that I'm not confused. I worked on some of the basics in January/February, but I would like to return to that.
  • I want to gain some software engineering skills. I'm decently comfortable with C++, and I would like to learn more broad design principles to take my programming to the next level.
